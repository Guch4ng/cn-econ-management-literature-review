# 中文经济管理文献综述 Skill

这是一个面向中文经济管理研究写作的 Codex skill，用于撰写、扩展、重构和审查文献综述。它强调文献真实性、综述逻辑、句级引用匹配和 GB/T 7714 参考文献格式。

## 适用场景

- 中文期刊论文的文献综述部分
- 博士论文、硕士论文和学位论文中的文献综述章节
- 国家社科、教育部、人文社科、地方课题等项目申报书的研究现状部分
- 公司金融、产业经济、数字经济、创新创业、管理学、政策研究、经济与管理等经济管理等方向主题
- 需要严格核验文献真实性、避免虚构文献和错引文献的学术写作
- 需要区分“综合性判断句引用”和“具体研究句引用”的综述写作
- 需要控制同一篇文献在全文中只引用一次的高强度引用规范

## 核心能力

这个 skill 不只是插入参考文献，而是帮助把文献综述从“文献罗列”改成“问题导向的综合论证”。

它会要求写作者先明确研究主题、学科场景、综述对象和写作用途，再围绕概念、理论、机制、方法、证据、分歧和研究缺口组织内容。

## 引用规则

- 正文采用顺序编码制，如 `[1]`、`[2]`、`[3]`。
- 文末参考文献采用 GB/T 7714 格式。
- 每一篇文献在正文中只能引用一次。
- 每一个引用编号必须对应文末唯一一条参考文献。
- 每一条参考文献必须真实存在、可核验，并且能够直接支撑其所在句子。
- 不得编造作者、年份、题名、期刊、卷期页码、出版社、DOI 或 URL。

## 句级引用方式

综合性判断句用于概括多个研究共同支持的理论路径、机制分类、研究结论或争议焦点，通常在句末集中放置多个未重复使用的引用。

具体研究句用于描述一篇特定文献的样本、方法、数据、结论或贡献，通常一句对应一篇文献。

## 综述写作标准

- 不按“某某研究了什么”机械堆叠文献。
- 每段应承担明确功能，如概念界定、理论发展、机制分类、方法比较、证据综合、分歧解释或缺口识别。
- 纵向上说明研究如何演进，横向上比较不同理论、方法、对象、数据和情境。
- 避免空泛表达，如“具有重要意义”“非常重要”“学者们进行了大量研究”等。
- 避免没有文献支撑的绝对化判断，如“已经证明”“一致认为”“毫无疑问”等。

## 使用方式

提示词：

```text
使用这个能力模块时，可以直接复制下面这段提示词：

请使用中文经济管理文献综述能力模块，围绕我给定的研究主题撰写一段结构清晰、逻辑严密、语言规范的中文文献综述。写作时必须严格核验文献真实性，不得编造作者、题名、期刊、年份、卷期页码、出版社、网址或其他参考文献信息。所有事实性陈述、理论观点、研究结论、机制解释、比较性判断和数据性表述，都必须由真实、可核验且内容匹配的文献支撑。正文引用采用顺序编码制，格式为[1]、[2]、[3]，文末参考文献采用GB/T 7714格式。每一篇文献在全文中只能引用一次，不能重复使用同一篇文献支撑后文相似或相关论述。对于综合性判断句，应使用多个不同文献共同支撑归纳性观点；对于具体研究句，应让一条文献准确支撑一个具体研究发现。文献综述不能写成简单的文献罗列，应当纵向梳理研究演进，横向比较不同理论、机制、方法、对象、证据和争议，最终提炼已有研究的不足和后续研究空间。请使用正式、专业、克制的中文学术表达，避免空泛词语和没有证据支撑的绝对化判断。如果某个论断缺乏足够可靠文献支持，应弱化表述、调整结构或明确提示需要补充文献，不得为了篇幅或完整性虚构引用。最后请输出文献综述正文和参考文献表。
```

在 Codex 中请求：

```text
请使用中文经管文献综述 skill，围绕我的主题撰写结构清晰、文献真实、引用不重复、符合 GB/T 7714 的文献综述。
```

然后提供研究主题、写作用途、已有文献或希望覆盖的研究范围。
