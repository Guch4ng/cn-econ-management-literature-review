# 中文经济管理文献综述 Skill 介绍

## 这是什么

`cn-econ-management-literature-review` 是一个面向中文经济管理研究写作的 Codex skill，主要用于辅助撰写、扩展、重构和审查文献综述。它不是简单的“参考文献插入工具”，而是一套围绕文献真实性、综述逻辑、句级引用匹配和 GB/T 7714 格式建立的学术写作工作流。

这个 skill 的核心目标是帮助使用者写出更严谨、更有结构、更符合中文学术规范的文献综述，尤其适合经济学、管理学、公司金融、产业经济、数字经济、创新创业、政策研究、体育经济与管理等方向。

## 它能做什么

这个 skill 可以帮助完成以下任务：

- 根据研究主题搭建文献综述框架。
- 区分概念界定、理论发展、机制解释、实证证据、方法比较和研究缺口。
- 将零散文献整理为有逻辑的综述结构。
- 避免文献综述变成简单的“某某研究了什么”的机械罗列。
- 设计综合性判断句和具体研究句的引用方式。
- 按顺序编码制在正文中插入引用，如 `[1]`、`[2]`、`[3]`。
- 按 GB/T 7714 格式整理文末参考文献。
- 检查文献是否真实存在、是否能够支撑对应句子。
- 控制同一篇文献在全文中只引用一次，避免重复引用和引用堆砌。
- 识别缺少文献支撑、过度概括、因果表述过强或引用错配的问题。

## 它解决什么问题

很多文献综述容易出现三个问题。

第一，文献堆砌。文章只是依次介绍不同作者的研究，缺少对研究脉络、理论关系和争议焦点的综合。

第二，引用不严谨。正文中的判断和文末参考文献之间缺少明确对应，甚至存在文献不真实、信息不完整、错引或过度引申的问题。

第三，综述逻辑不清楚。读者看完之后仍然不知道这个领域的核心概念是什么、主要解释路径有哪些、已有研究解决了什么、尚未解决什么。

这个 skill 的设计就是为了减少这些问题。它要求在写作前先建立文献与论点之间的对应关系，再围绕研究问题组织文献，而不是先写一段文字后再随意补引用。

## 关键写作原则

这个 skill 强调两类句子的引用方式。

一类是综合性判断句。它用于概括多个研究共同支持的理论路径、机制分类、研究结论或争议焦点。这类句子通常在句末放置多个不同来源的引用，但每个来源都必须真实、可核验，并且直接支持该句。

另一类是具体研究句。它用于介绍某一篇特定文献的样本、方法、数据、结论或贡献。原则上，一句具体研究句对应一篇文献，避免将单篇文献的发现扩大为普遍结论。

同时，这个 skill 保留一条较严格的规则：同一篇文献在全文中只能引用一次。如果后文需要支持相近论述，应寻找其他真实可核验的文献，或调整文章结构，减少重复表述。

## 使用后的效果

使用这个 skill 后，文献综述应达到以下效果：

- 结构更清楚，读者可以快速理解研究主题的主要脉络。
- 逻辑更深入，不只是总结别人做了什么，而是解释不同研究之间的关系。
- 引用更严谨，每个关键判断都有明确文献支撑。
- 参考文献更规范，正文编号与文末条目一一对应。
- 表达更学术，减少空泛、口语化和无实质信息的表述。
- 研究缺口更明确，能够自然引出后续研究问题或论文贡献。

## 适用场景

这个 skill 适合以下场景：

- 中文期刊论文的文献综述部分。
- 博士论文、硕士论文中的文献综述章节。
- 课题申报书中的国内外研究现状、研究基础和文献评述。
- 经济管理类论文的理论基础、机制分析和研究假设部分。
- 需要把已有文献从“资料堆积”整理成“逻辑综述”的写作任务。
- 需要严格核验参考文献、避免虚构文献和重复引用的学术写作任务。
- 需要按照 GB/T 7714 输出中文参考文献格式的正式文本。

## 不适合的场景

这个 skill 不适合用于不需要文献核验的快速随笔、观点短评或非正式写作。它也不适合在没有任何文献来源、又不能联网或不能提供文献材料的情况下强行生成带参考文献的综述。

如果可核验文献不足，这个 skill 会倾向于缩小表述范围、标注文献缺口，或建议补充文献，而不是为了让文字看起来完整而编造引用。

## 一句话总结

这个 skill 的作用是：帮助中文经济管理研究写作者在不虚构、不错引、不堆砌文献的前提下，写出结构清楚、逻辑深入、引用规范、能够支撑论文问题意识的高质量文献综述。
